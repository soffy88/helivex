## §6 Verdict

### Overall: **NO-GO** — no filter enables gate pass on any instrument

**Root cause: PBO, not DSR.** HMM already achieves positive DSR (BTC +0.417, SOL +0.594), meaning the signal clears the multiple-testing significance bar. The blocker is PBO=1.00: the CPCV fold with the best IS Sharpe is consistently the worst OOS fold. This is a data-quantity artifact — with 12 months and 75% chop, trending periods (Oct 2025 bull, Dec 2025–Jun 2026 bear) are concentrated in specific calendar windows, not spread across folds. Any fold that gets the trending window as IS also gets the choppy window as OOS → IS rank ≠ OOS rank → PBO=1.0.

With 3–5 years of 4H data (~13–22k bars), trend periods would be distributed across multiple folds and PBO should drop below 0.5. The alpha is real — DSR > 0 confirms it; CPCV just can't validate it on 12 months.

**Path forward: extend data to 3+ years.** OKX `history-candles` 4H is available back to 2019 for BTC/ETH. With longer history: use HMM filter (best segment performance) → expect DSR to hold positive and PBO to drop.

Best filter by mean DSR across instruments: **HMM 2-state** (DSR = 0.337)
